# Phase 2: Basic CRUD Operations

Implement core database operations:
- Put() for single triple insertion
- Put() for batch triple insertion (slice)
- Get() with pattern matching on subject/predicate/object
- Del() for single and batch triple deletion
- Query creation with automatic index selection
- Support for limit, offset, and reverse options
- Basic filtering support with callback functions
- GenerateBatch() utility function
